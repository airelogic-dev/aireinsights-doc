    Pipeline triggers

    Jul 2, 2021

    Trigger definition

    Name of the trigger

    The name has to be unique (accross one user) and is used to reference the trigger to update or delete it.

    yaml

    Name: example_trigger

    json

    "name": "example_trigger"

    Data collection

    This references the data collection the data should be taken from.

    yaml

    DataCollection: my-data

    json

    "dataCollection": "my-data"

    Pipeline

    The pipeline that should be invoked by the trigger.

    yaml

    Pipeline: my-pipeline

    json

    "pipeline": "my-pipeline"
